LocalForage Hooks and Providers for React.

## Usage

With npm(or yarn, or pnpm):

```bash
npm install react-forage
```

In `.jsx` / `.tsx` or other files:

```ts
import { LocalForageProvider, useLocalForageState } from 'react-forage'
```
