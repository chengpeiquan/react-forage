React Forage

LocalForage Hooks and Providers for React.

Get Started

Installation

Install the module with your favorite package manager:

Terminal
npm install react-forage

Examples

In the following example, state data will be stored in IndexedDB. You can switch to the Application panel in the browser console (press F12). In IndexedDB, you can see a database named react-forage-website and a data table named examples.

Store state into IndexedDB
